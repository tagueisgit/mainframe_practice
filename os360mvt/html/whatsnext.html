OS/360 on Hercules: What's next?

12. What's next?

There are still several bugs to be fixed:

  • TSO logons get S305 ABENDs if more than 8 MB of memory is in the system. (Possibly, if the TSO region winds up below 8 MB, it might work even if more than that is defined, but I don't know a good way to find out.) For now, this problem can be avoided simply by defining only 8 MB, but a fix would be a Good Thing.
  • Earlier builds of this system encountered an S806 ABEND in HOSRDR. I don't know what happened to fix this problem, and it might reappear.
As we come across other things that need fixing, they'll be noted here as well.

I strongly recommend that you join the hercules-390 mailing list if you're interested in this system. As we find and fix bugs, that's the place they're noted first. You can subscribe via the Yahoo! Groups hercules-390 group page. Another mailing list, hercules-os360, is dedicated specifically to OS/360 on Hercules, and is much lower volume.
